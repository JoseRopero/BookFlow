# BookFlow

Aplicación móvil para gestionar una biblioteca digital de libros electrónicos, permitiendo a los usuarios buscar, descargar y leer eBooks.

## Características
- Registro e inicio de sesión de usuarios.
- Búsqueda y filtrado de eBooks.
- Descarga y lectura de eBooks.
- Gestión de reservas y penalizaciones.
- Sistema de reseñas y calificaciones.

## Instalación
1. Clona el repositorio: git clone https://github.com/tuusuario/bibliotecadeebooks.git
2. Abre el proyecto en Android Studio.
3. Sincroniza las dependencias con Gradle.
4. Configura Firebase:
- Agrega el archivo `google-services.json` al directorio `app/`.

## Uso
- Ejecuta la aplicación en un dispositivo o emulador Android.
- Regístrate o inicia sesión para comenzar a utilizar la aplicación.
